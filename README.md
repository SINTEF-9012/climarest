# ClimaREST Mussel Farming Demo

## Prerequisites

```sh
$ mamba create --name climarest python=3.11
$ mamba activate climarest
$ mamba install streamlit xarray matplotlib plotly numpy pandas cartopy
```

## Development

```sh
$ cd ./src
$ streamlit run app/main.py
```
